Дата и время — Datum & Uhrzeit
Как говорить о датах и времени
Официальное время (цифры): 7.15 = sieben Uhr fünfzehn. Разговорное время: halb drei = половина третьего (2:30), Viertel nach zwei = четверть третьего (2:15), Viertel vor drei = без четверти три (2:45).

Страны и национальности — Länder · Nationalitäten
Страны, жители, прилагательные
Схема: страна → житель (м/ж) → прилагательное. Пример: Türkei → Türke / Türkin → türkisch • Russland → Russe / Russin → russisch • Deutschland → Deutscher / Deutsche → deutsch • Frankreich → Franzose / Französin → französisch. Россия особо важна для Ивана: Russland, der Russe / die Russin, russisch.
